Собираться схема будет здесь.

Тестовый запуск пока включает в себя:
1. последовательный авто-проход по элементам (игнор контейнеров на 1-м уровне).
2. для рефлекса понимает команду "продолжить с другого элемента": перейти на {id: N}
3. контейнеры в корне программы "игнорируются авто-проходом" и им присваивается статус модульный: TAG_NAME модульный
4. для рефлекса понимает команду "заинклудить модульный контейнер": подключить {id: N}
p.s. значение N заменять на число, например: подключить {id: 23}
p.s. максимум вложенных инклудов: 3.
p.s. максимум зациклиенных переходов: 10.
